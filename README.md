# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses [Oxc](https://oxc.rs)
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) uses [SWC](https://swc.rs/)

## React Compiler

The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see [this documentation](https://react.dev/learn/react-compiler/installation).

## About The Project

	This is my Assignment 6 project. I built this website to show digital tools in a clean way where users can browse products, add items to cart, and check out easily. I focused on making the UI simple, responsive, and easy to use.

## Key Features

--> Responsive Multi-Section Landing Experience:
    Real-time Order Summary with dynamic total price and includes hero, stats, workflow, steps, pricing, and footer sections All sections are responsive for mobile, tablet, and desktop devices.
 
--> Modern Pricing Architecture: 
    A transparent, three-tier pricing model (tarter, Pro, and Enterprise) designed with responsive UI elements to build trust and clarity.

--> Smart Add to Cart and Checkout Flow:
    One selected item cannot be added two times at a time. If user tries to add the same product again, it shows an "Item already in cart!" toast message. Also, when user clicks "Proceed To Checkout", all selected cart items are cleared and a success message is shown.
   
--> Interactive Cart System:
    Users can add and remove products from cart and see updates instantly ,duplicate Item Prevention in Cart and Toast Notification for cart actions.
